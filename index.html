Installing

  1. Download and run the installer.

  2. Click Next.

  3. Enter an Installation Folder (ASCII, no accents, spaces, or symlinks, short path).

  4. Click Next.

  5. Tick Run MSYS2 now.

  6. Update the package database and Pacman with:

    pacman -Sy pacman

  7. Close MSYS2, and start it again by running MSYS2 Shell from the Start Menu. Update the package database and core system packages with: 

    pacman -Syu
    After running this command, some errors may appear, this is normal. After the command completes, close MSYS2. If a dialog appears that warns you that processes are currently running, it's fine to ignore it and close MSYS2 anyway.

  8. At this point, the location of the MSYS2 bootstrapping script has changed, so the Start Menu entries will need to be fixed so that they point to the correct script. Open up the file location of the MSYS2 Start Menu entry. On Windows 10 you can do this by right clicking on the MSYS2 Shell Start Menu entry, selecting More, and then selecting Open File Location. Once the file location appears, right click on the MSYS2 Shell shortcut and select Properties. Delete everything under Target and replace it with something like: 

    C:\msys64\msys2_shell.cmd

    You might have to change the previous location slightly depending on where you installed MSYS2 and whether you downloaded the 32-bit or the 64-bit installer. When in doubt, check for yourself to make sure the script exists.

    Press OK and then change the Target for the MinGW-w64 Win64 Shell to: 

    C:\msys64\msys2_shell.cmd -mingw64

    Lastly, change the Target for the MinGW-w64 Win32 Shell to:

    C:\msys64\msys2_shell.cmd -mingw32

  9. Open MSYS2 again and run:

    pacman -Su
