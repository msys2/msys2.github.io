msys2 installer

One click installer for msys2

  msys2-i686-20140910.exe   msys2-x86_64-20140910.exe View on GitHub

  1. Download and run an .exe - msys2-x86_64 for 64bit, msys2-i686 otherwise

  2. Click "Next":

  3. Enter Installation Folder (ascii, no spaces nor symlinks, short path):

  4. Tick Run MSYS2 now:

  5. Update the system packages with Pacman -Syu:
    Then close MSYS2 and restart the shell

  6. Now Pacman is fully committed to the windows cause :)

  7. Take look at Detailed MSYS2 install guide for additional details and troubleshooting.
